### Introduction

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.1.3, then upgraded to 18.


Prerequisite : 
- `nvm use 18` -- node v18.19 or higher required
- `npm install`
Check ng version : `[npx] ng --version`

Please prefer to use the local angular cli by using npx.

### Development server

- `[npx] ng serve` --> `http://localhost:4200/`
The application will automatically reload if you change any of the source files.

### Build

`[npx] ng build`
The build artifacts will be stored in the `dist/` directory.
